# cladding-configurator

React/Vite proof of concept for roof/eave cladding miter geometry.

Features:

- Direct corner-angle input.
- Jobsite measured-angle mode: measure two legs from the corner and the diagonal between them; the app derives the corner angle.
- Top-down miter preview.
- Fabrication dimensions for piece A and piece B.

## Run locally

```bash
npm install
npm run dev
```
